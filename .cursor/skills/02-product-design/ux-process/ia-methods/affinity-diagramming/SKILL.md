# Affinity Diagramming

## When to Use

- Synthesizing discovery research
- Grouping observations, quotes, ideas
- Aligning team on IA themes before structure

## Process

1. **Generate** — Write each observation/quote on a separate note
2. **Group** — Organize notes by theme without labeling first
3. **Name** — Name each theme once grouped
4. **Prioritize** — Identify 3–5 most critical themes
5. **Extract** — Insight statements: what does this tell us about users?

## Output

- Thematic clusters with labels
- Insight statements
- IA implications

## Synthesis Template

Use [synthesis-template.md](synthesis-template.md) for handover format.

## Downstream Use

- Feeds IA themes and category labels
- Informs Card Sort (open) item selection
- Converts to HMW questions for design
