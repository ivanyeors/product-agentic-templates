# Information Architecture Methods

Select the right IA method based on phase, context, and goal. Each method has a subskill with SKILL.md and synthesis/handover template for agent use.

---

## Method Selection Framework

| Phase / Context | Goal | Method | Subskill |
|-----------------|------|--------|----------|
| Pre-IA, existing product | Inventory and assess content | Content Audit | [content-audit/](content-audit/) |
| Discovery synthesis | Group research findings, align team | Affinity Diagramming | [affinity-diagramming/](affinity-diagramming/) |
| New IA, no structure | Discover mental models | Card Sort (open) | [card-sort-open/](card-sort-open/) |
| Existing IA to validate | Test proposed structure | Card Sort (closed) | [card-sort-closed/](card-sort-closed/) |
| Navigation validation | Test findability before design | Tree Testing | [tree-testing/](tree-testing/) |
| Navigation validation | Test first-click decisions | First-Click Testing | [first-click-testing/](first-click-testing/) |
| IA output | Define hierarchy and labels | Site Map Structure | [site-map-structure/](site-map-structure/) |
| Complex content, metadata | Controlled vocabulary, tagging | Taxonomy Design | [taxonomy-design/](taxonomy-design/) |
| Content-heavy product | Define content types, fields | Content Model | [content-model/](content-model/) |

---

## Typical IA Process Flow

1. **Content Audit** (if existing product) → inventory and assess
2. **Affinity Diagramming** (if discovery synthesis) → thematic clusters
3. **Card Sort (open)** → discover mental models, category labels
4. **Site Map Structure** or **Taxonomy Design** or **Content Model** → define structure
5. **Card Sort (closed)** or **Tree Testing** or **First-Click Testing** → validate

---

## Subskill Outputs and Downstream Use

| Method | Output | Feeds Into |
|--------|--------|------------|
| Content Audit | Inventory, quality assessment, recommendations | IA Document, Site Map |
| Affinity Diagramming | Thematic clusters, insight statements | IA themes, HMW questions |
| Card Sort (open) | Category labels, groupings | Site Map Structure |
| Card Sort (closed) | Placement accuracy, confusion matrix | IA revision |
| Tree Testing | Success rate, directness, time | IA revision, findability report |
| First-Click Testing | First-click accuracy, confusion points | Entry-point recommendations |
| Site Map Structure | Hierarchy, depth/breadth rules | IA Document, artifacts-template |
| Taxonomy Design | Controlled vocabulary, facets | IA Document, metadata |
| Content Model | Content types, fields, metadata | IA Document, Backend schema |
