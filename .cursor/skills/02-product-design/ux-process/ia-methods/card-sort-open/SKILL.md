# Card Sort (Open)

## When to Use

- New product, no existing structure
- Need to discover mental models
- Content or navigation structure is ambiguous

## Process

1. **Prepare** — 40–80 items (cards). User-facing labels. No predefined categories.
2. **Run** — 15–20 participants. Remote (Optimal Sort, Maze) or in-person.
3. **Analyze** — Similarity matrix, dendrogram. Identify common groupings.
4. **Synthesize** — Category labels + item groupings.

## Output

- Category labels (user-generated)
- Item groupings per category
- Similarity matrix summary

## Synthesis Template

Use [synthesis-template.md](synthesis-template.md) for handover format.

## Downstream Use

- Feeds Site Map Structure (category labels, hierarchy)
- Informs IA Document navigation model
