# Card Sort (Closed)

## When to Use

- Proposed structure exists
- Need to validate category assignments
- Measure user agreement with IA

## Process

1. **Prepare** — Items + predefined categories (from open sort or IA draft).
2. **Run** — 10–15 participants.
3. **Measure** — Percentage placing each item in expected category.
4. **Analyze** — Confusion matrix. Identify problematic items.

## Output

- Placement accuracy per item
- Confusion matrix (items × categories)
- Validated vs. problematic categories

## Synthesis Template

Use [synthesis-template.md](synthesis-template.md) for handover format.

## Downstream Use

- Feeds IA revision (relabel, regroup problematic items)
- Informs Tree Testing (if structure validated)
