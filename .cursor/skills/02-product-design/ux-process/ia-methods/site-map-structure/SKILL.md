# Site Map Structure

## When to Use

- After Card Sort (open) or Content Audit
- Defining IA hierarchy
- Output for IA Document

## Conventions

- **Depth:** Max 3 levels for primary content
- **Breadth:** 7±2 items per level maximum
- **Labels:** User-facing language, not internal labels

## Output

- Hierarchical structure (mermaid or tree)
- URL structure (if web)
- Section purposes

## Handover Template

Use [handover-template.md](handover-template.md) for IA Document format.

## Downstream Use

- Feeds IA Document Site Map section
- Feeds artifacts-template IA Document
- Informs Tree Testing (validation)
