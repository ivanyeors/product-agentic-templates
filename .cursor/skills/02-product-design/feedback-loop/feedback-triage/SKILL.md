# Feedback Triage

## When to Use

- Feedback channels are planned
- Need to define how feedback feeds roadmap
- PM prioritization process

## Process

1. **Rubric** — Criteria for prioritization (impact, frequency, effort).
2. **Escalation** — When to escalate to stakeholders.
3. **Cadence** — Triage frequency (weekly, sprint).
4. **Document** — Triage rules for handoff.

## Output

- Triage rubric
- Escalation rules
- Handover to backlog

## Handover Template

Use [handover-template.md](handover-template.md) for handover format.

## Downstream Use

- Referenced in handoff package
- Informs PM backlog prioritization
- Part of Feedback Channels Plan handover
